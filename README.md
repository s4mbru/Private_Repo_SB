# Finch Artist
This repo allows a user to download our content and be able to use a Finch 2.0 Robot to draw abstract art

Includes a Frontend folder with a singular page (index.html), a Backend folder with the BirdBrain Python library for Mac as well as a "private.py" script to run the backend. The "app.py" is the script that runs the website and connects the frontend and the backend. It is intentionally left as a template to be filled out in class.
